The Castles of Dr. Creep

Launch the game from the shortcut
Select which castle you want to play.
Any castles found will be displayed.
The intro will begin, scrolling through the castle rooms.

Controls:
Press Right Ctrl for a 1 player game.
Press Numpad 0 for a 2 player game.
Press F3 to display the Highscores

Controls:
Press Enter to return to the intro
The map screen displays rooms which have been visited.

To start:
Player1 Press Right Ctrl
Player2 Press Numpad0

Game Position:
Press F1 to Save The Game
Press F2 to Load A game
